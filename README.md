# JSAB - Just Shapes & Beats

A rhythm-based dodge-'em-up game built with vanilla HTML5 Canvas and JavaScript. Inspired by *Just Shapes & Beats*.

## How to Play

- **WASD** or **Arrow Keys** to move
- **Space** to dash through hazards
- Survive until the end of each level

## Features

- 2 levels with unique boss fights
- 3 difficulty modes (Normal → Hard → Nightmare)
- Procedurally generated music based on BPM and melody
- Local co-op support
- Multiple hazard types: projectiles, beams, walls, pulsing rings, waves

## Difficulty System

| Mode | Speed | Spawns | Player HP | Unlock |
|------|-------|--------|-----------|--------|
| Normal | 70% | 65% | 5 | Default |
| Hard | 100% | 100% | 3 | Beat Normal |
| Nightmare | 140% | 145% | 2 | Beat Hard |

## How to Run

Open `JustShapesAndBeats.html` in any modern web browser. No server or build tools required.

## Controls

| Action | Player 1 | Player 2 |
|--------|----------|----------|
| Move | WASD | Arrow Keys |
| Dash | Space | Enter |
